# AI-and-computer

This repository serves as a knowledge base and code collection for my exploration into Medical AI, Healthcare Data Science, and Computational Medicine. As a medical student, my goal is to leverage programming and AI to solve real-world clinical problems, moving from "black box" algorithms to interpretable medical applications.

# Key Focus Areas
Clinical Decision Support: Building agents and models to assist in diagnosis and treatment planning.

Medical Data Analysis: Processing and visualizing EHR (Electronic Health Records) and medical imaging data.

LLMs in Medicine: Fine-tuning Large Language Models for medical Q&A, report generation, and literature review.

Algorithmic Thinking: Applying computer science fundamentals to medical problem-solving.

# Tech Stack & Tools

Languages: Python, SQL, R

AI/ML Frameworks: PyTorch, TensorFlow, Scikit-learn, Hugging Face Transformers

Medical Tools: MONAI (Medical Open Network for AI), BioPython, OHIF Viewer

Deployment: Streamlit, FastAPI, Docker

Disclaimer: This project is for educational and research purposes only. The code and models provided here are not intended for direct clinical use or patient diagnosis without further validation.
